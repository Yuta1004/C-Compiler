## Commands

```
// Run
cd compiler
make
vim program.c 
./yncc program.c > tmp.s
gcc -ni-pie -o tmp tmp.s
./tmp
echo $?

// Test(C)
cd compiler
make test

// Test(SH)
cd compiler
make test-sh
```
